TFL Colours

Implementation is really simple, just give what ever you want to color-code a two classes.

Install instructions can be found on the Github repo readme. Meanwhile here are a few examples...

Example 1 - Simple Usage

  • tfl
  • tfl-{underground-line}
        <p>Take the <span class="tfl tfl-bakerloo">Bakerloo</span> Line</p>

Take the Bakerloo Line

Example 2 - With Bootstrap

When you add some more CSS on top, like Bootstrap, it really comes alive...

        <p>Take the <span class="badge tfl tfl-bakerloo">Bakerloo</span> Line</p>
